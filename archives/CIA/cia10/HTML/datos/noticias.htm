- Debian saco la version 2.2 basada en el kernel 2.2.16, trae instalacion grafica y soporte PCMCIA incorporado (pa portatiles)

- Corel saco el Corel Linux OS Second Edition

- 3dfx el fabricante de tarjetas graficas para juegos esta trabajando en controladoras Linux para la Voodoo5 55000 de 64MB

- Suse Linux saco la version 7.0 en español de su SO. Incluye cosas mejoradas (por ej. la herramienta de instalacion grafica YaST2 y el reconocimiento automatico de hardware). Incluye en el paquete 200 herramientas de pre-KDE y mas de 1500 herramientas Linux actualizadas. (por ej, StarOffice 5.2.). Tb algunas cositas mas profesionales para usar servidores web, de correo, proxy y noticias en Linux. La documentacion son 4 manuales para usuarios principiantes. pvp unas 10.000 ptas +iva (unos 50$) 60 dias de asistencia gratuita para la instalacion. http://www.suse.de/es.

- Red Hat Linux 7.0 ya en la calle!! ABC Analog saco la version 7.0 de Red Hat. Entre las novedades: una version del kernel 2.4 y aplicaciones y librerias preparadas para soportarlo cuando salga la version oficial de este kernel, soporte para dispositivos USB. Necesita minimo: 32 MB ram, y 500 MB HD libre. En esta version son 15 CDs con: sistema, fuentes, soft, juegos, aplicaciones y documentacion. El mas barato 7400ptas. http://abcnet.es
