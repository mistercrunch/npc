Instalacion de los sistemas operativos

- Linux

Esta es la primera instalacion que deberiamos hacer, y vamos a usar el Lilo (Linux Loader) para bootear todo. El Lilo tiene que estar en los primeros 1024 cylinders para que labure bien.

Hace una particion de 1.5gb e instalale el linux. eso deberia bastar para jugar bastante, cualquier soft adicional puede ser agregado en otra particion. Esta particion de linux tambien tiene que ser la "activa", no vamos a enseñar a instalar el linux en esta ocacion asi que vas a tener que arruinar las particiones par de veces hasta que te salga si lo estas haciendo por primera vez.


- Windows 98 (SE)

El segundo en la lista es Windows 98. Use el fdisk del 98 porque en versiones anteriores (95 y dos) no aguanta el tamaño de este rigido. Los cd's del 98 son booteables asi que no vas a tener drama en comenzar desde ahi. Nosotros queremos tener acceso a esta particion desde Windows NTtambien, asi que tiene que ser FAT 16 (NT 4.0 no puede ser fat32). Cuando corras el fdisk eleji [N]O en el comienzo (sino lo haces te va a hacer la particion en fat 32). Ahora hace la particion primaria de dos, la cual NO debe ser una particion activa.

Despues de que la particion este hecha,formateala, rebootea y edita el Lilo en /etc/lilo.conf. Agrega hda2 como win/dos

Despues corre el instalador del Windows 98 normalmente.

- Windows NT 4.0

de nuevo, usa el FDISK desde el cd del Windows 98 . Hace una particion secundaria con tres drives logicos. una particion de 2000mb (sera de NTFS, pero hacela en fat16 por ahora) para NT 4.0, una de 2000mb (despues va a ser de fat16) y una de 9000mb (de NTFS) .

Instala el Windows NT 4.0 sobre el primer drive logico. Los boot-files para Windows NT seran instalados en el C: (Windows 98) . Estos archivos van a ser llamados: NTDETECT.COM, BOOT.INI and NTLDR.

copia estos archivos al directorio: C:\ntdetect\nt4. Despues que el NT se ha instalado, usalo para formatear las otras particiones que quedan. asi que, la particion de 2000mb sera formateada en fat16 y la particion grande (de 9000mb) sera formateada como NTFS.

- Windows 2000 Professional

Bootea con Windows 98 y mete el cd del Windows 2000 ,el setup te va a preguntar si queres upgradear, selecciona nueva instalacion (lo que significa que no upgradees!). Usa el espacio que sobra para hacer una particion y meter el 2000, hace otra NTFS e instala normalmente. Nuestro Windows 2000 quedara en el H:

Copia el NTDETECT.COM, BOOT.INI and NTLDR desde el Windows 2000 al directorio C:\ntdetect\win2000.


El Boot.ini

Editaremos el Boot.ini manualmente con un editor, porque este no se incluira en el windows 2000 despues de su instalacion.

Asi es como debe quedar:

[Boot Loader]
Timeout=5
Default=multi(0)disk(0)rdisk(0)partition(4)\WINNT

[Operating Systems]
multi(0)disk(0)rdisk(0)partition(4)\WINNT="Windows NT Workstation 4"
multi(0)disk(0)rdisk(0)partition(4)\WINNT="Windows NT Workstation 4 [VGA mode]" /basevideo /sos
multi(0)disk(0)rdisk(0)partition(3)\WINNT="Windows 2000 pro"
C:\="Microsoft Windows 98"

Aun, es imposible cambiar entre Windows NT 4.0 y Windows 2000. El NTDETECT.COM del NT 4.0 y del 2000 difieren en tamaño, asi que vas a tener que cambiar manualmente el archivo este en el C: .hace backups!
