NUEVO TROYANO MIRC/GIF
----------------------
Ha aparecido un nuevo troyano llamado mIRC/Gif, diseñado para propagarse 
a través el chat, mediante el cliente de IRC "mIRC".

Este troyano destaca por su capacidad para disfrazarse de forma casí 
perfecta como un fichero de imagen tipo "gif" 

El método que utiliza para aparentar ser un gif, lo consigue debido a 
que posee la cabecera y extensión de los archivos de imagen gif.

Este troyano no tiene capacidad de enviarse automáticamente sino que 
hay que enviarlo de manera voluntaria, como si del Backorifice o del 
Netbus se tratara.

Una vez enviado el troyano a la víctima, esta tiene que escribir en su 
cliente de irc la siguiente línea: 
/load -rs c:\mirc\dcc\images\foto.gif
donde foto.gif es el nombre que le habéis puesto al troyano junto con 
la extensión gif

Una vez que la víctima haya ejecutado la orden, le aparecerá una
pantalla de error perteneciente al "mIRC".

Una vez infectada, el troyano creará en el directorio de Windows un
fichero llamado Himem32.sys que contiene una copia del virus. Además
de que genera una entrada en el archivo mirc.ini para que el troyano
se ejecute cada vez que se inicie la aplicación "mIRC".

El puerto utilizado por este troyano es el 64000.

ENCONTRADO UN VIRUS QUE PROMETE PORNOGRAFIA GRATIS PARA INFECTAR A LOS USUARIOS
------------------------------------------------------------

El virus conocido como Aphex o Aplore se esparce por la red con la promes de porno gratis.
Este virus toma el control del navegador y los clientes de chat.
El virus llega al usuario con un archivo adjunto de la extension .com.
Al ejecutarse este archivo, abre un puerto 8180 en nuestro pq y hace que escuche como
servidor web.

El virus crea una pagina llamada index.html e hace que todo aquel que conozca nuestra
ip y se conecte a ella se encuentre con la pagina web.
En la pagina no unico que se consigue es infectar a un nuevo usuario. 

Una vez infectado nuestro ordenador el virus debe propagarse, por lo que abre un servidor IRC
y manda mensajes a todos los usuarios de diversos canales ofreciendo pornografia gratuita. Junto
con este mensaje se envia un enlace a nuestra ip. Que envia al usuario a la pagina infectada.

Este virus no es un virus muy dañino pero si muy molesta, ya que si el infectado es alguien que 
lo recive por ejemplo en el trabajo, debe disculparse por molestos mensajes, etc...

Si el usuario usa AOL insant messenger, cualquier mensaje que mande tendra un atachment con
el mensaje de siempre, porno gratis.


De nuevo vemos con esto lo vulnerable que es American On Line.

SOLO NACER, UNA VULNERABILIDAD DEL OFICE DE XP
-----------------------------------------------


Nada mas empezar con el windows xp ya nos hemos encontrado de una importante vulnerabilidad en
su software. El experto en encontrar fallos George Guninski, es quien ha encontrado este agujero.
El primero es el Office Spreadsheet, una pequeña hoja de calculo creada para su destinacion
en paginas web.
Posee una error en la funcion de host que pertmite que todos aquellos
usuarios con malas intenciones puedan tomar el control de nuestra computadora.
También hay otro agujero localizado en el programa Outlook conocido ya por MUCHOS  agujeros en 
el software. Esta vez el programa permite que en un e-mail hecho en html contenga scripts 
maliciosos. Que se activan cuando la victima del mensaje malicioso intenta responder o reenviar
el mensaje que contiene el script.
Microsoft, antes de que ellos tengan que hacer nuevos cd's y gastarse dinero en la nueva version,
recomiendan a los usuarios que se desactive el soporte para html de outlook. Y que tampoco se
use el editor Word como editor de correo. Guninski da otra mejor solucion, cambiar todo el 
paquete informatico por otras soluciones.
