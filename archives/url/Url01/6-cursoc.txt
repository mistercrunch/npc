#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[])
{
  
  system("PAUSE");
  return 0;
}

Este es el proyecto base que crea devcpp. ¨Que es todo eso? Compiladlo y ejecutarlo( CTRL + F10 ). Os saldra una pantalla 
pidiendo un nombre de archivo para el fichero de texto que veiamos. Dale el nombre que quieras pero acabando en .c por ejemplo, "test.c" .
Como veis no hace absolutamente nada excepto mostrar una pausa. esto solo es el esqueleto para que funcione tu programa. Examinemoslo un poco.

  system("PAUSE");

Esto es la funcion system. Una funcion es un conjunto de ordenes que llamamos por un nombre. en el caso de system lo que hacen esas ordenes es ejecutar un comando de sistema. Aqui ejecuta el comando PAUSE para que el programa no finalize hasta que queramos.Como veis la linea acaba con punto y coma ( ; ) que es como deben acabar todas las sentencias dentro de una funcion menos algunas sentencias condicionales y los bucles

#include <stdio.h>
#include <stdlib.h>

Estas sentencias incluyen los ficheros de cabecera necesarios para poder usar alguna funciones. explicare esto con mas detalle mas adelante.

int main(int argc, char *argv[])

Esta es la funcion main, que es a la que llama el sistema operativo cuando ejecuta el programa. Todos los programas para MS-DOS deben tener una funcion main y los de windows una funcion WinMain. Como veis despues de esta linea se abren unas llaves  "{" y al final del programa se cierran "}". estas llaves siven para delimitar las ordenes que estan "dentro" de main, es decir, las que  ejecutara el sistema operativo. Los argumentos int argc y char *argv[], los veremos tambien mas adelante.

  return 0;

Esto simplemente le devuelve (return) un codigo numerico ( 0 ) al sistema operativo para poder comprobar errores en el programa. 

Antes de que empieces a escribir codigo te advertire que c es un lenguaje en el que no se interpretan igual las mayusculas y las minusculas. No es lo mismo MAIN que main o Main. Como les gusta decir a muchos es "case-sensitive". Tened mucho cuidado entonces en escribir sentencias correctamente. Ahora vamos a hacer un poco mas interesante el programa, modifica el programa para que se vea asi.

#include <stdio.h> //Cabeceras de las funciones
#include <stdlib.h>

/*Codigo principal*/
int main(int argc, char *argv[]) //Funcion main
{
  printf("Hola mundo\n"); //Imprime hola mundo
  system("PAUSE");   //Pausa antes de terminar.
  return 0;
}


Como podeis ver, este programa imprime la mas que tipica frase "Hola mundo". Podemos ver algunas novedades.

Podemos ver como se introducen comentarios en un programa. Todo lo que este dentro de /* y */ sera ignorado por el compilador al procesar el archivo. Asimismo, todo lo que este despues de //, sera considerado un comentario.

La otra novedad es la funcion printf. esta funcion imprime una cadena en pantalla. Una cadena es una secuencia de caracteres delimitada por comillas como "Hola" o "32". ¨ Y como hacemos que pase a la siguiente linea ? Pues ponemos el codigo de escape \n en la cadena. el caracter de escape '\n' hace que printf pase a la siguiente linea. Probad a quitarla y compilad y ejecutad 
el programa.

Un codigo de escape es una barra invertida \ seguida de una letra o una combinacion de numeros que nos permite poner algunos caracteres especiales en las cadenas.
estos codigos son:
\a Campana  
\b Retroceso
 \f Avance de linea
\n Nueva linea 
\r Retorno de carro 
\t Tabulacion horizontal 
\v Tabulacion vertical 
\' Comillas simples
\" Comillas dobles 
\\ Barra invertida
\ooo Codigo ASCII del caracter en octal
\xhhh Codigo ASCII del caracter en hexadecimal 

//Ejemplo 3

#include <stdio.h> 
include <stdlib.h>


int main(int argc, char *argv[]) 
{
int number;
number=3;
printf("El numero es %d\n",number); 
system("PAUSE");   //Pausa antes de terminar.
return 0;
}

Ahora vemos como se declara una variable ( int number ). Una variable es un espacio en la memoria donde puedes meter valores. A esta variable se le pone un nombre ( number ) y se decide que tipo de datos se van a albergar en ella (int). El tipo de datos int es un numero entero.  Despues de declararla le asignamos un valor ( number = 3 ).
Tambien podeis inicalizarlas ( asignarles un valor inicial ) a la vez que la declarais:

int number=3;

o declarar varias en una linea:

int number,numero;

o declarar e inicializar varias en una linea.

int number=2, numero=3;

Debo remarcar que debemos inicializar todos las variables. Nunca debemos utilizar variables sin inicializar, asi que es buena idea inicializarlas al declararlas aunque sea con valor 0.

En la funcion printf vemos %d. Esto sirve para poder imprimir el valor contenido en una variable. En este caso, substituye %d por la variable number. Tambien podriamos poner mas variables.
//Ejemplo 4

#include <stdio.h> 
include <stdlib.h>


int main(int argc, char *argv[]) 
{
int number=2;
int numero=3;
int suma =number + numero;
printf("%d+%d=%d\n", number, numero, suma);
system("PAUSE");   //Pausa antes de terminar.
return 0;
}


Como podeis ver las variables se toman en el orden en que se las damos, es decir, el primer %d es substituido por number y el segundo por numero. Asi sucesivamente podeis imprimir todas las variables que necesiteis.
Podeis observar un nuevo operador cuyo significado es obvio ( + ). Aqui teneis algunos operadores aritmeticos:

Operador             Significado
     +                       Suma
     -                       Resta
     *                  Multiplicaci¢n
     /                     Division
