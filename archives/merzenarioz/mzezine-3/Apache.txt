En este peque๑o articulo voy a explicarles como montar un servidor web como es el caso de Apache 

HTTP Server. Si, aunque no lo crean existe una version para windows de este exelente y popular servidor usado

permite trabajar con MySQL, PERL y PHP


Para conseguir este servidor web solo tenes que descargarlo de la siguiente URL :

}{}{}{==  http://internautas.gueb.net/servidores/WEB/apache_1_3_12_win32.exe   ==}{}{}{

Aproximadamente pesa unos tres megas. Una vez finalizada la carga solo tenes que ejecutarlo y si lo dejas en

http://www.microsoft.com/windows95/downloads/contents/WUAdminTools/S_WUNetworkingTools/W95Sockets2/Default.asp

Configuracion///////////
จจจจจจจจจจจจจจจจจจจจจจจจจจจ    
Una vez que tenemos el server instalado tenemos que configurarlo, para esto tenemos que ir al directorio 'conf' y 

abrir, usando el notepad o cualquier editor de textos, el archivo 'httpd.conf'. Al abrirlo solo hay que editar un par 
de lineas y sera suficiente para que quede bien configurado.

Busca casi en el principio el siguiente fragment0:
_______________________________________________________________________
### Section 1: Global Environment
#
# The directives in this section affect the overall operation of Apache,
# such as the number of concurrent requests it can handle or where it
# can find its configuration files.
#
ServerName TU.IP.VA . ACA  <---|||||Tenes que agregarle esta linea|||||
#
# ServerType is either inetd, or standalone.  Inetd mode is only supported on
# Unix platforms.
#
_________________________________________________________________________

Como saber tu IP? si estas desconectado va a ser siempre 127.0.0.1 y si estas conectado a internet andate a 
inicio-->ejecutar----> y pone 'command' . Despues pone 'ipconfig',dale enter y listo.

Para tener una IP estatica podes ir a cualquier site que te de servicio de Static DNS [dnsalias.org]

Una vez que tenemos configurado el ServerName solo falta configurar el RootDocuments para que ya este 

funcionando a la perfeccion.

Busca en el mismo texto la siguiente parte:
______________________________________________________________
# DocumentRoot: The directory out of which you will serve your
# documents. By default, all requests are taken from this directory, but
# symbolic links and aliases may be used to point to other locations.
#
DocumentRoot "C:\Archivos de programa\Apache Group\Apache\htdocs"  <---|||||Modifica este directorio||||||
_______________________________________________________________

Si no lo modificas tu directorio raiz va a ser 'htdocs', osea si vas a poner una pagina web, pones el index.htm 

dentro de la carpeta htdocs y ya tenes la pagina principal:-). Cambia el directorio a tu gusto, por ejemplo:


Ej: " C:\Mp3" :-D

Una vez que terminamos de hacer estas dos modificaciones, guardamos el texto y nos vamos a 

inicio>>>programas>>>Apache Group>>>>'Start Apache'

Listo!! poniendo en el explorador tu IP veras tu pagina web o el listado del directorio que hayas puesto como 

Root.

Solo resta decir que se puede seguir editando el httpd.conf y personalizarlo aun mas....pero con esto ya estas 

listo para poner a correr tu propio servidor web.
