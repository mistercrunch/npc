Introducción
============

El Telnet es una eficaz herramienta que podemos encontrar en Windows o Unix
El finger es un servicio empleado para ver si un usuario se encuentra conectado
en un servidor como nuestro buen amigo el root (súper usuario - (administrador))
como también se puede saber cuales son todos los usuarios conectados a ese mismo servidor
Le voy a explicar como es en el sistema operativo Windows, abrimos y utilizamos
el Telnet pero no de la forma tradicional, esta vez, se lo va hacer por el puerto
79 que es el del servicio FINGER, hay un problema, si el servidor no tiene habilitado
este puerto o listening estamos medios jodidos y para buscar mas información habría
que probar otra forma de adquirirla pero imaginemos que lo tiene habilitado. :)

Practica:
=========
 
En Windows desde el prompt del msdos en la carpeta Windows escribes:

c:\windows> telnet servidor.com 79 
(o completas los datos en el programa de TELNET)

en unos segundos y estarás conectado a la victima pero no se presentará nada,
solo una pantalla en blanco luego si yo quiero saber acerca de el root escribo 
root, no podré ver lo que escribo así que deben escribir bien una vez que ya
he escrito root presiono enter y el me mostrara los datos disponibles de ese 
usuario (una  vez  que el sistema te proporciona los datos pierdes conexión con el
mismo asi que copia lo que necesites) lo  cual puede ser de mucha ayuda para
tener acceso al sistema ya que muchos usuarios colocan su nombre o sus iniciales
con el año  como  sus passwords por lo que se puede jugar con esa información a
ver que tal si por el contrario quiero saber de todos los usuarios conectados
al sistema le das en lugar de root le colocas @ y asi te muestra todos los usuarios

servidor.com:
es el servidor al cual quieres tener acceso o donde se conecta la victima
no tiene que ser justamente .com puede ser cualquier cosa.
