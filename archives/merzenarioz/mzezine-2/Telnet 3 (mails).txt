Introducción
============
Lo que voy a explicar es utilizar una técnica que se le puede encontrar alguna
utilidad, como hacer bromas o para utilizarlo como ingeniería social..., etc. 
es la del MAIL ANONIMO.
No es muy complicado, pero hay programas que simplifican este tipo de técnica
pero me parece bueno que sepan como hacerlo por TELNET.
En este ejemplo le mando un mail a una empresa diciéndole que soy el director
de ARNET... (jejeje, ni yo me la creo , pero buee, es un ejemplo)


Practica:
=========
En Windows desde el prompt del msdos en la carpeta Windows escribes:

c:\windows> telnet servidor.com 25 
(o completas los datos en el programa de TELNET) (como siempre, jeje)
el puerto 25 corresponde al de correo salida SMTP cuando conectemos
nos saldrá un mensaje


220 Howdy From (elservidor) running Smail 3.1.30.16 (ESMTP)
[d/o/u/g] ready at Mon, 8 dec 01 00:01:30 -300 (CDT) 

(algo parecido, no exactamente igual...)

Hay un comando que muchas veces sirve en los servidores que es "help" 
y te van a salir todos los comandos, también se puede escribir "echo" y la
maquina nos contestara que esta preparada para empezar.

Para redactar un e-mail anónimo hay que seguir los siguientes pasos...


mail from: arnet@servicios.net

(aqui se pone quien lo envia (en este caso el ARNET))

El servidor responde...

250 <arnet@servicios.net> ... Sender Okay

Ahora tecleamos lo siguiente:

rcpt to: victima@arnet.com 

(a quien se lo enviamos o sea la victima :)

El servidor responde...

250 <victima@arnet.com> ... Recipiant Okay 

para continuar se escribe:

data

354 Enter mail, end with "." on a line by itself  (o algo similar)

To: victima@arnet.com
From: arnet@servicios.net
Subject: Aviso

Estimado Usuario:

Esta es una situación muy incomoda.
Como director de ARNET el mayor  servicio
de ARGENTINA, me sentí abrumado al descubrir
un robo de nuestra bases de datos que tuvo
lugar el fin de semana 5 de agosto del 2001
Incluida en la información perdida 
estaban entre otras cosas los datos personales
de un pequeño porcentaje de nuestros clientes.
Aunque su nombre, por suerte no estaba en la base
de datos, sigue existiendo alguna inseguridad.
En consecuencia, le pedimos que nos mande los
siguientes datos: Nombre, Dirección, Código postal,
Teléfono, Contraseña antigua(invalida)
Contraseña Nueva (actualizada).
Le descontaremos el 50% de su cuenta de Septiembre.
ARNET, el mayor servicio de INTERNET en ARGENTINA,
le garantiza que los datos previos serán ingresados
antes del 1 de Septiembre y que serán confidenciales
antes y después de la fecha.


. 

El punto sirve para terminar el mail (perdon por que el mail es tan largo, pero
les puede servir para algun chamullo que se les ocurra : )


El servidor responde...

250 Mail accepted 

Para finalizar esto se escribe:

quit 

y listo!, no era tan dificil como pensaba, jejeje..


Consejo muy obvio
=================
Bueno, esta forma de mandar e-mails anónimos no es 100% segura, ya que no es completamente
anónima, ya que algunos servers registran tu IP, asi que hay que tener cuidado!
te recomiendo que uses una cuenta Shell y también un PROXY, pero tene cuidado que
tampoco es 100% seguro como parece.
Hay algunos programas que simplifican estas cosas y son muy fáciles de usar, pero nunca
esta demás saber estas cosas :)
