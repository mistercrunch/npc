  • Motorola StarTAC 7760
    •

 

  1. <FCN> 0000000000000 <RCL> (Ingreso al modo de programación).
  2. Enter Phone Number: Ingresar MIN del celular : 547050….<STO>
  3. Enter Country Code: 722 <STO>
  4. Enter Network Code: 33 <STO>
  5. Enter AMPS System ID: 32128 (Córdoba) <STO>
  6. Enter CDMA System ID: 32128 (Córdoba) <STO>
  7. Enter CDMA Network ID: 65535 <STO>
  8. Enter CDMA Primary Channel, System A: xx <STO>
  9. Enter CDMA Primary Channel, System B: 384 <STO>
  10. Enter CDMA Secondary Channel, System A: xx <STO>
  11. Enter CDMA Secondary Channel, System B: 777 <STO>
  12. Enter Secure Code: 000000 <STO> (valor por defecto)
  13. Enter Lock Code: 123 <STO> (valor por defecto)
  14. Enter Priority Number 1: 0 <STO> (valor por defecto)
  15. Enter Priority Number 2: 0 <STO> (valor por defecto)
  16. Enter Priority Number 3: 0 <STO> (valor por defecto)
  17. How many Phone numbers: 1 <STO> (Finaliza programación de NAM 1 y sale del modo de programación, Mensaje ‘Entry Done’).

 

Observaciones

XX no interesa, dejar el valor por defecto.

Verificar que el terminal esté en banda B: <RCL> <*> Seleccionar Scan B <STO>

El cambio de vocoder rate de 13 K (valor por defecto) al valor 8 K EVRC (valor a utilizar por CTI) se realiza por intermedio del Menú Test (menú oculto).

 

  • Para habilitar el Menú Test, realizar los siguientes pasos:
  1. Ingresar al modo test: <FCN> 00**83786633 <STO> (El display muestra US’)
  2. Ingresar a la NAM con: 55#
  3. Avanzar con * hasta el paso 09 (el n° de paso se indica en la parte superior izquierda del display)
  4. En el paso 09 ingresar: 01000000
  5. Avanzar con * hasta finalizar ( ´ )
  6. Salir del modo test con 01# (re-inicio automático del equipo)
  • Para ingresar al Menú Test presionar: <FCN> <FCN>.
  1. Avanzar con la tecla de volumen lateral hasta el parámetro VOCODER
  2. Cambiar el VOCODER preferido con la tecla SMART ó SND al valor 8 K EVRC
  3. Salir del Menú Test con la tecla <END>.
  • Luego deshabilitar el Menú Test con los siguientes pasos:
  1. Ingresar al modo test: <FCN> 00**83786633 <STO> (El display muestra US’)
  2. Ingresar a la NAM: 55#
  3. Avanzar con * hasta el paso 09 (el n° de paso se indica en la parte superior izquierda del display)
  4. En el paso 09 ingresar: 00000000
  5. Avanzar con * hasta finalizar ( ´ )
  6. Salir del modo test con 01# (re-inicio automático del equipo)
  7. Verificar la inhabilitación del Menú Test: <FCN> <FCN> no afecta

 

  • Samsung SCH-411

 

1. Digitar código de ingreso al modo de programación: 47*869#08#9. Ingresar al menú principal del modo de programación.
Opciones Menú:

  1. General
  2. Setup NAM1
  3. Setup NAM2
  4. Setup NAM3
  5. Setup NAM4

2. Seleccionar 1: General.

3. Programar parámetros:

  • Para pasar del parámetro actual al siguiente, utilizar la tecla lateral de volumen.
  • Para seleccionar opciones posibles, usar * ó #
  • Para grabar opción,  utilizar la tecla OK
  • Para volver al menú anterior utilizar CLR
  1. ESN = 9F031….. (Número de Serie Electrónico del terminal N/A).
  2. CAI version = 2 (Versión de Interfaz de aire común soportado p/el móvil N/A).
  3. VOC8/13K/EVRC = SO_VOIC_EVRC <OK>.
  4. SCM = 01101010 (N/A)
  5. Lock Code = 0000 (N/A)
  6. Slot Mode = Yes <OK>
  7. Slot Index = 2 <OK>
  8. Pref NAM (1-4) = Digital pref <OK>

4. Presionar <CLR> para regresar al menu principal del modo de programación.

5. Seleccionar 2: Setup NAM 1.

6. Ingresar al menú de programación de la NAM 1.
Opciones Menú:

  1. Phone #
  2. FM
  3. CDMA

7. Seleccionar 1: Phone #

8. Programación de parámetros

  • Phone N° = 547050….. <OK>
  • Mobile ID # = 547050... <OK>

9. Seleccionar 2: FM

10. Programar parámetros:

  1. FM Home SID = 32128 <OK> (Córdoba)
  2. FM 1st Chn = 334 <OK>
  3. FM Acq SID 1 = 0 <OK>
  4. FM Lock SID 1 = 0 <OK>
  5. Auto Reg = Yes <OK>
  6. FM pref…. = B only <OK>

11. Seleccionar 3: CDMA

12. Programar parámetros:

  1. IMSI_MCC = 722 <OK> (Country Code).
  2. IMSI_NNC = 33 <OK> (Country Code).
  3. CDMA pref.. = B only <OK>
  4. Pchn Sys B = 384 <OK>
  5. Schn Sys B = 777 <OK>
  6. CDMA Home SID = Yes <OK>
  7. CDMA fSID = Yes <OK>
  8. CDMA fNID = Yes <OK>
  9. SID #1 = 32128 <OK> (Córdoba).
  10. NID #1 = 65535 <OK>

11. Seleccionar <END>, para finalizar la programación y reiniciar el terminal.

 

  • Hyundai HGC-110

 

1. Digitar código de ingreso al modo de programación: #86904**89397#.

2. Ingresar al menú de programación.
Opciones de Menú:

  1. NAM 1
  2. NAM 2
  3. NAM 3
  4. NAM 4

3. Utilizar las teclas * ó #, seleccionar el item correspondiente: (NAM 1) y luego <MEM>

4. Ingresar al submenú de NAM .
Opciones de Menú:

  1. NAM NAME
  2. PHONE NUMBER
  3. FM SETUP
  4. CDMA SETUP

5. Seleccionar PHONE NUMBER.

  • Enter MOBILE PHONE NUMBER = 547050... <MEM>

6. Seleccionar FM SETUP

  1. Seleccionar 1ST PAGING = 334 <MEM>
  2. Seleccionar HOME SYSTEM ID = 32128 (Córdoba)
  3. Seleccionar ACQUISITION SYSTEM NUMBER = 32128 (Córdoba) <MEM>
  4. Seleccionar AUTO REGISTRATION = YES <MEM>

7. Volver con CLR al submenú NAM 1.

8. Seleccionar CDMA SETUP

  1. Seleccionar MCC # = 722
  2. Seleccionar MNC # = 33
  3. Seleccionar 1st CH B = 384
  4. Seleccionar 2nd CH B = 777
  5. Seleccionar HOME REGIS = YES <MEM>
  6. Seleccionar ACQUIR SYS = 32128 (Córdoba) <MEM>
  7. Seleccionar FOREGN SID = YES <MEM>
  8. Seleccionar FOREGN NID = YES <MEM>
  9. Seleccionar SID – NID: SID # 1 = 32128 (Córdoba) <MEM> ; NID # 1 = 65535

9. Volver con CLR al submenú NAM 1.

10. Con CLR volver al Menú ppal

11. Presionar END para finalizar la programación y apagar del equipo.

12. Encender el equipo.

13. Seleccionar MENU 8 1 MODO PREFER = CDMA > FM <MEM>

14. Seleccionar MENU 8 2 1 PREF CDMA = SOLO SYS B <MEM>

15. Seleccionar MENU 8 2 2 PREF ANALOGO = SOLO SYS B <MEM>

16. Seleccionar MENU 9 1 EVRC MODE = YES <MEM>
