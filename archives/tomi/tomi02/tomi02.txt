--[02]------------------------|[Curso de HTML]|---------------------<§> by SySt3m <§>---- 
                               
   
                                  

  -Bueno esta es mi primera guia, asi que debe tener algunos errores
   ya que soy novato en esto,pero le haremos empeño, tratare de hacerla
   facil y sencilla.En esta guia, explicare como hacer una pagina web
   en formato HTML y con algunos comandos JAVASCRIPT, ya basta de lata
   y vamos a la accion.


1.-Estructura de una documento HTML.
 
  -Un HTML empieza con <html></html>, dentro de estos se especifican 
   dos zonas <head></head> y <body></body>
  -Entonces para empezar a escribir un HTML esto seria asi:

<HTML>
<HEAD>
<TITLE> aqui ponen el titulo de su web</TITLE>

<BODY>


</BODY>
</HTML>

1.1 -Comandos basicos de un HTML 

  -Estos son los comandos mas estandar que destacan:

a)- TITULOS

  -Van encerrados entre los marcadores <H#> y </H#>, siendo # un numero 
   entre 1 y 6, donde 1 es el titulo mas grande:

<H1>mi primera pagina web</H1> mi primera pagina web
<H2>mi primera pagina web</H2> mi primera pagina web
  etc....

b)-Ordenar la posicion del texto

  -Ya,  para que el texto valla en el centro seria asi:

<CENTER> texto </CENTER>                         
  -Para que el texto valla a la derecha  seria asi:

<RIGHT> texto </RIGHT>                               
  -Para que el texto valla a la izquierda seria asi:

<LEFT> texto </LEFT>

c)-Salto de linea

  -Para esto se ocupa <BR> es un comado unitario( no tiene su termino 
   de comando) produce una nueva linea.

d)-Salto de parrafo

  -Pa esto se ocupa <p> solo, ya que es suficiente pa que haga la orden 
   este comado produce una linea y un salto de linea extra. 

e) Efectos en el texto

-<B></B> da el efecto de negrita al texto
-<I></I> da el efecto de letra italica
-<em></em> da el mismo efecto que la italica
-<STRONG></STRONG> da el mismo efecto que la negrita  
  
-Otro  efecto es el color,tamaño y tipografia esto seria asi

  <FONT SIZE="">este comado da el porte a la letra.Entre las comillas 
   pones el tamaño ej:
size=1
size=+1
size=2
size=+2
etc....

  *aclaracion tb el porte de la letra puede ser - o sea me explico onda
   asi
size=-1
size=-2
size=-3
size=-4
etc....

   COLOR="" aqui puedes poner el color en ingles o en hexadecimal
  *acuerdate de cerrar con </FONT> :)
  *otra cosa mas adalante escribo una lista de numeros en hexadecimal.

  -Entonces lo que explique quedaria asi:
 
<FONT SIZE="tamaño de la letra COLOR="el color">el texto </FONT>

  -Tambien se puede determinar una clase especifica de letra, ese
   comando es el siguiente:

<FONT FACE="aqui especificai la letra">
 
f)-Listas

  -Existen 3 tipos de listas: las numeradas. sin numerar y de
   descripcion.
  
  -Numeradas:

<OL>
<LI>mi
<LI>primera
<LI>web
</OL>
  
  -Este comado hace lo siguiente :

1. mi
2. primera
3. web

  -Sin numerar:
<UL>
<LI>mi
<LI>primera
<LI>web
</UL>
  
  -Este comando hace lo siguiente
   
   mi
   primera
   web

-Una lista de descripcion:
<DL>
<DT>primero (<I> primero </I>)
<DD>antes del segundo :)
<DT>segundo (<I> que puede ser aer.. </I>)
<DD>despues del primero jejej
</DL>
-Este comando quedaria de la siguiente manera :
 primero ( primero)
   antes del segundo :)
segundo (que puede ser..)
   despues del primero jejej
 
g)- Tablas

  -El comado es:

<TABLE></TABLE>
  -Las tablas se construyen a base de filas y columnas pa hacer una
   columna <TD></TD> y pa las filas <TR></TR>
  -Tambien se le asigna un ancho (WIDTH) que va en % osea entre 0% a 100%, tambien 
   se le puede agrega un borde(BORDER) que sirve pa que se vea mas ordenado, a este borde 
   le podemos asiganar un color ,tambien se puede escojer el tamaño de las celdas(CELLPADDIG) 
   y hacer un espacio entre celda osea una me tida en otra(CELLSPACING)( a estas tambien se 
   le puede poner color)).Ademas si quieres puedes ponerle un color determinado a cada culumna 
   (<TD BGCOLOR=" ">) o acada fila (<TR BGCOLOR=" ">) 
 -Se puede especificar el lugar de la tabla con el parametro ALIGN="left,center o right" 


-Su sintaxis seria haci: 

<table width="%" border=" " cellspacing=" " cellpadding=" " align= " "bgcolor=" " 
bordercolor=" "> 
</table> 

ejemplo: 

<TABLE width="95%" border="1" cellspacing="1" cellpadding="2" aling="center" bgcolor="#ffffff" 
bordercolor="#ff0080"> 
<TR> 
<TD> PRIMERA </TD> 
<TD> SEGUNDA </TD> 
<TD> TERCERA </TD> 
</TR> 
<TR> 
<TD> CUARTA </TD> 
<TD> QUINTA </TD> 
<TD> SEXTA </TD> 
</TR> 
</TABLE> 

h)-Imagines

  -El comado es:

<IMG src="ubicaion de la foto" border="el borde entre 0 y 10"  width=
"ancho" height="largo">

  -Para especificar la URL del archivo de imagen a desplegar se ocupa 
   <IMG src="URL">
  -Para ponerle un nombre a la foto se ocupa ALT"nombre o descripcion" esto sirbe por 
   si no se visualiza la imagen 

  -Podemos ademas poner la pocicion del texto al lado de la foto arriba abajo etc... 
   para eso ocupamos el parametro ALIGN seguido de: 

TOP: pone el texto en la parte mas alta de la imagen 
MIDDLE: alinea el texto en la mitad dela imagen 
BOTTOM: pone el texto en la parte mas baja de la imagen 
LEFT: pone el texo en la parte izquierda de la imagen 
RIGHT: pone le texto en la parte derecha de la foto 


i)-Fondos y Colores: 
  -Podemos hacer que el fondo de nuestra web sea de algun color o alguna 
   foto gif.Esto seria asi:

BGCOLOR : Indica el color de fondo
BACKGROUN : Indica la imagen de fondo

-Tambie podemos decir los colores que tendra el texto y los link.

  TEXT  : Indica el color del texto en la pagina.
  LINK  : Indica el color del link sin visitar.
  VLINK : Indica el color del link ya visitado.
  ALINK : Indica el color del link cuando se esta markando
          (siempre es rojo) :( 

-Un ejemplo de esto seria asi:

<body bgcolor=" " text= " " link= " " vlink=" " alink " ">

j)- Atributos de HR

  ALIGN   : Alinea miento de la linea ( center, left o right )
  SIZE    : Grosor de linea 
  WIDTH   : Ancho
  NOSHEDE : Pone linea solida

Ejemplo:
Mi
<hr size 10 >
Primera 
<hr noshade>
Web
<hr align= left width=50%>

  *Se lo puede agrecar color a la linea con tan solo poner color=" " 
   ejemplo :

Mi
<hr size 10 color= red>
Primera 
<hr noshade color = blue>
Web
<hr align= left width=50% color = yellow>

k)- Tag

  U   : Para subrayar
  SUB : Para poner subindice
  SUP : Para poner  indicide

  Ejemplo:

3<sub>2</sub>
2<sup>3</sup>
<u> mi primera pagina web </u>

l)- Frames:

  -Se usa para dividir la pantalla.
  -Los Tag que se usan son:
  <FRAMESET> : Define un conjunto de frames ya sea horizontal (COLS) o 
   vertical (ROWS)
  <FRAME> : Con este Tag se especifica el nombre del Frame y la URL
   que va a desplegarse en el.
  <NOFRAME> Sirve para delimitar texto que sera ingnorado por los Browser
   que acepten Frames: Se cierra con </NOFRAME>
 
  Ejemplo Vertical:

<frameset rows ="50%,50%">
<frame src = " " name="arriba">
<frame src = " " name="abajo">
</frameset>

  Ejemplo Horizontal

<frameset cols ="15%,33%,*">
<frame src = " " name="izquierda">
<frame src = " " name="derecha">
</frameset>
  
  Ejemplo Horizontal y Vertical:

<frameset rows ="16%",*>
<frame src = " " name="arriba">
<frameset cols ="30%",*>
<frame src = " " name="indice">
</frameset>
</frameset>

n)- Hojas de estilo
 
  Wueno que wea son las hojas de estilos y pa que nos sirven, esta 
  hojas nos permiten tener mayor control sobre nuestro html ya que
  podemos especificar muchos atributos como son colores, margenes,
  tamaño de letra y artas weas mas.Podemos ocupar las hojitas estas, 
  como paginas cabronas, perdon maestras es decir que muchas paginas 
  pueden tener el mismo aspecto, estas hojas pueden ser creadas por dos 
  tipos de sintaxis, CSS (Cascade Style Sheets) y JavaScript.Wueno despues 
  de esta pequeña intro demosle. 

  
Como hacer estas hojitas:

 En CSS los valores y propiedasdes se encierran entra llaves {}, cada 
 propiedad es seguida por dos puntos y el valor tambien. Cada par propiedad/valor 
 esta separado  por un punto y coma (;). ejemplo:

<style type="text/css">
<!--
     p {font-weight: bold; color: red;}
     h1 {align: center;}
-->
</style>

  Esto hace que que cuando pongamos un parrafo <p> lo va aponer del 
  estilo negrita y rojo y tambien que cuando pongamos un titulo h1 lo
  va a poner al centro.

  * los estilos se pueden encerra en comentarios (<!-- ... -->), 
    de esta forma los navegadores que no cachen que wea es <style> 
    la ignoren.                                                                
  **Los CSS son re complicados onda se te olvida un punto y coma y 
    culaquier wea, no te va a funcionar na asi ke si algo que salga en 
    esta guia no te funka me mandas un email(ya que soy humano y me puedo
    equivocar)
 
  Ahora lo dire como hacer en lo JavaScript 

  Pa hacer lo mismo ke explique con los Css se hace asi :

<style type="text/javascript">
<!--
   tags.p.weight ="bold";
   tags.p.color ="red";
   tags.h1.aling = "center"
-->
</style>

   *Si no cachan que son las tag loreen un poko mas arriba( los tags 
    son las weas ke le hacemos aun texto onda color tipo de letra 
    etc...), una aclaracion no puse document.tags ya que no es necesario
    ya que los tags simpre se aplican a los documentos.
  **El encabezamiento simpre va haci <script language="JavaScript"> y 
    se cierra con </scrip>


Tambien se puede ocupar with ejemplo:

with (tag.p)
{
       color = "red"; 
       fontWeight = "bold";
       fontStyle = "italic"; 
}

   Yap aer ke mas...aah una wea importantate por ejemplo si ponen esto:

<body>
    <h1>¡ToMi TeAm la <em>Ezine</em> diferente!</h1>
</body>
    
y decean dejar el titulo rojo y  hacen esto :

<style type="text/css"> h1 {color: red;} 
</style> 
<body> 
<h1>¡¡¡ToMi TeAm la <em>Ezine</em> diferente!!!</h1> 
</body> 

   -Y luego cachan que les quedo todo rojo eso pasa porke <html>es 
    cabron con el <body> y este asu vez es cabron con el <h1> y el 
    <h1> es cabron con <em> eso quiere decir que al poner tan solo el
    estilo de <h1> el <em> lo adopta en otras palabras es copion, pa esos 
    casos tienen que especificar en <style> el color en este caso de <em> 
    eso seria haci
  
<style type="text/css"> h1 {color: red;} 
                        em {color: blue;}
</style> 
<body> 
<h1>¡¡¡ToMi TeAm la <em>Ezine</em> diferente!!!</h1> 
</body> 

   Yap como esto ya podran hacer hojas de estilo es cosa que vallan
