OPCIONES de BitchX:(/help para ver los comandos; /help comando para la accion)
 
    Para entrar en un servidor: /server irc.???.es 6667 (u otro puerto)

    Para entrar en otro server estando conectado ya en uno es de la forma:
    /server +irc.???.es puerto (en este caso solo tendras una pantalla para
    manejarte por toda esa red, no lo podemos tener todo :-)

    Para cambiarte el nick: /nick otro nick

    Para entrar en un canal: /join #canal o /j #canal (da lo mismo)

    Para salir de un canal: /part (tienes que estar en la pantalla del canal)

    Para saber kien es algun usuario: /whois nick

    Para saber ls IP de un usuario: /dns nick

    Para hacer un privado: /query nick

    Para hacer un dcc: /dcc chat nick (Una vez que te han aceptado el dcc,
    tienes que comenzar la conversacion, esto se realiza kon la opcion:
    /query =nick y ya podras hablar con el)

    Para ver la gente del canal hay varias opciones:
	Si estas bajo una shell normal:  F2 - Marca los usuarios normales
					 F3 - Marca los usuarios operadores
	Si estas bajo XWindows: /sc - Marca todos los usuarios. Esta opcion
	tambien sirve para la consola normal.

    Para realizar una accion sobre un canal: /me texto

    Para kickear a alguien: /kick #canal nick (lo mismo para el ban)

    Para realizar un ping sobre un usuario: /ctcp ping usuario

    Para enviar ficheros por dcc: /dcc send nick /directorio/archivo

    Para ignorar a alguien: /ignore nick tiempo (puedes ignorarlo un tiempo 
    limitado que eso lo pondras despues del nick o si no pones nada lo ignoras
    hasta que lo designores)

    Para designorar a alguien: /unig nick

BitchX es un Script multiventana,pero no va igual k ninguno de Windows, asi
que para cambiar de ventana tienes que activarla lo primero, primero lo 
explicare y luego pondre un ejemplo para k se vea mejor.

    Con la opcion /window new hide activas una ventana, asi k si yo pongo
  /window new hide 2 ya tendre activada esa ventana y podre utilizarla, el
  mismo caso es para las demas.
    Para cambiar de ventana se hace: Presionando la tecla Esc (escape), luego
  la sueltas y presionas el numero de ventana (si trabajas en una shall normal
  esto tb. se puede hacer presionando la tecla Alt+numero_de_ventana sin
  soltar la tecla Alt, es decir yo presiono la tecla Alt y al mismo tiempo
  presiono el numero de ventana y ya habre cambiado a otra ventana siempre que
  este activa)

Ejemplo: Ya he entrado en mi servidor y por ahora solo tengo la ventana 1 que 
es la principal activada. Tecleo /window new hide 2 y se me activa la vantana
2 y ahora para cambiar a esa ventana presiono la tecla Esc, la suelto y 
presiono la tecla 2 y ya habre cambiado a esa ventana, ahora ya solo tengo k 
entrar en el canal k kiera o hacer un query o lo k me de la gana.

Esto es bien sencillo, para cambiarte el nombre que va a aparacer simplemente

Ahora a partir de todo esto, voy a ense¤ar algun trukito para no tener ni k
especificar el nick ni toda la informacion del whois ni el server para entrar,
asi si lo tienes especificado anteriormente para esta aplicacion ya no hace 
falta ponerlo de nuevo, kon tal de arrankar el BitchX por la forma sencilla ya
tienes todo configurado, lo k tienes que hacer es lo siguiente:

  En el home del usuario, es decir si yo arranko kon usuario "One", pues en
  el directorio /home/One habra k hacer:

	a.- Editar un archivo .bitchxrc y poner:
		IRCNAME el_k_kieras
		IRCUSER tmb_lo_k@te_de_la.gana
	Con esto conseguimos especificar el nombre y host que te aparecera en
	tu /whois.

	b.- Editar un archivo .irc_server y poner:
		irc.lander.es
		irc.ddnet.es
		irc.communet.es
	Aqui especificaras el server o los servers k vayas a poner por default
	para entrar, iran por orden, primero intentara entrar en el primero, y
	asi sucesivamente. Me parece k komo maximo tienes que poner tres server
	pero eso no es seguro asi k tu prubalo de todas formas.
	NOTA: Si has especificado ya los servidores en el include/config.h no
 	hace falta poner este archivo de nuevo.

	c.- Y por ultimo en el .profile del usuario (segun la shell k tengais)
	tienes que poner:
		export IRCNICK="tu_nick"
	Asi definiras el nick k vayas a utilizar en las chat y no tendras que
	ponerlo.
