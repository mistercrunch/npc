Introducción:
-------------

Perl-> Practical Extraction and Report Language.
Es un lenguaje de scripts creado por Larry wall, con el objetivo de simplificar
las tareas de administrador de UNIX. Perl se ha convertido en un potente
lenguaje y ultimamente se utilaza mucho para la programacion de cgi's, en
paginas web. Si teneis experiencia con C,grep, awk o sed comprobareis que os
resulta familiar ya que hereda estructuras de estos lenguajes. Cuando hablemos
de Perl nunca usaremos la palabra programa con nuestras "creaciones", sino
scripts (ya que es un lenguaje interpretado, aunque internamente funcione como
un compilador). Para la realizacion de este cursillo es necesario que tengais
instalado perl  en vuestra maquina, yo voy a usar la version de linux , aunque
podeis usar la version de cualquier otra plataforma (es un lenguje muy
portable). Perl tiene licencia GNU (si, ya sabeis... distribucion de codigo
fuente, poder modificarlo, gratuidad...).


Primeros Pasos
---------------

La version que voy a usar es la 5.0 (aunque ya ha salido la 5.6)
Si no sabeis si lo teneis instalado, probad con:
#which perl   
/usr/bin/perl  -> si lo teneis instalado os dara un respuesta como esta

Para ver la version que teneis escribir:

#perl -v   ->esto os mostrara la version, licencia, copyright ... 

#perl -h   ->la ayuda...

tambien podeis sacar informacion mirando las paginas man...

#man perl

Como ireis comprobando Perl es muy flexible, y la mayoria de las cosas que se
os ocurran se pueden hacer de distinata forma.

Bueno vamos a aprender la primera instruccion. Vamos a sacar por pantalla el
tipico hola mundo.

#perl -e 'print "hola mundo\n";'

Bien, perl -e lo que hace es que la instruccion print se pueda poner en la
linea de comandos.
Print ->esta instruccion permite sacar por pantalla un mensaje.
\n hace un "retorno de carro" es decir que imprima la informacion y salte a la
linea siguiente.
; ->todas las lineas han de acbar con ;


Como os dije anteriormente, en Perl hay distinatas formas de hacer las cosas...
Esta script se podia ejecutar de otra forma
1º creas un archivo con la extension .pl  (ej: prueba1.pl)
2º con cualquier editor de textos le poneis la anterior
orden
print "hola mundo\n";
3º ejecutais el script con: 
#perl prueba1.pl

Si no podeis ejecutar el script acordaos de darle permisos... (chmod 755
prueba1.pl)

Como ya hemos realizado el primer script con exito, ahora vamos con el segundo:
El siguiente script lo que ará sera calcular el iva de un producto.

print "Que precio tiene el producto?\n";
$valor= <stdin>;
$iva= $valor * 0.16;
print "IVA=$iva\n";

Como veis en la segunda linea aparece una variable (como su propio nombre
indica es un valor que en cada ocasion varia), las variables se indican con el
simbolo $ , y cada variable se declara en el momento de usarla.
En este ejemplo tenemos dos variables: valor e iva.
<stdin> ->se refiere a la entrada estandar (el teclado). (lo contrario es
          <stdout> que es la salida estandar: el monitor)
En la 3ª linea aparece la segunda variable, que es el resultado de multiplicar
         (*) la primera variable por 0.16 (el iva).
En la 4ª linea sacamos por el monitor el resultado de la opercion (que es la
variable iva). Esto ultimo se llama interpolacion de variables que lo que viene
a significar es que una variable entre comillas sera substituida por su valor.
(el nombrecito este se usa para demostrar lo mucho que sabes, ...si es que el
mundo esta lleno de pedantes ....)
