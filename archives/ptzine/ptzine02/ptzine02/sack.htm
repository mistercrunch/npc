O s@ckp@ss consiste num pequeno executável feito em Delphi podendo ser feito em qualquer outra linguagem ( Basic, C, C++ ... ) mas porque não em Delphi .

O funcionamento do s@ckp@ss e muito simples e muito eficiente. Que vou passar a explicar agora:

1- Arranjamos um Lamu no IRC que queira jogar um jogo que seja via NET pois isso não e problema, que e uma maravilha e como ja se sabe são COLHOES deles, depois de encontrarmos o Lamu falamos com ele e perguntamos qual o jogo que este quer jogar na NET.

2- Depois falamos com o Lamu e dizemos que temos aqui um software "altamente" para jogar na net e que era mais simples se ele o tive-se também, ok mandamos a file compilada para o Lamu e dizemos para ele a correr que isso o irá ligar automaticamente ao server do jogo onde nos já estamos a espera mas na realidade não estamos queremos e sim apanhar a passwd do Lamu.

3- Ok ele correu o prog C00L só que a melhor parte ta vir aquilo deu um erro heheheh =). Irá dar: Erro de pass e login uma cena estupida =). Entao dizemos que estamos lá a espera dele hehehe só que enquanto isso dava o login dele e pass foram escritas numa file na dir C:\WINDOWS\SYSTEM então inventamos uma desculpa toda catita e dizemos ao Lamu para nos mandar a file que é a w95.dll onde se encontra o login e passwd dele heheheh =) só não ta simples de ser vista pois quem sabe se o Lamu nao tem uma ideia inteligente e decide abrir a file logo eu pus a passwd de certa maneira encriptada para só nos sabermos como sacar o login e pass do Lamu.

Ok, ja temos a file na nossa posse o que fazemos pa obter o login e pass heheheh =).

Simples:

Usamos o software aki encluido para desencriptar para obtermos o login e pass. =)
