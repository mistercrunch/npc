Aquí estoy otra vez con el segundo numero de Binary Zone. Seguramente mucho de vosotros pensabais que esto iva a ser una magazine como las demás, que aparece un numero y después nadie sabe mas de ellas. Pues lo siento mucho, pero no va a ser así. Aquí voy a estar mas y mas números para hacer que sufráis con esta revista ;)))En este numero tenemos muchas nuevas secciones, gracias a las colaboraciones de algunos lectores, y otras simplemente gracias a las recomendaciones de las encuestas. He recibido muchas cartas, y eso es de agradecer, pero necesito mas colaboraciones, así que si no tenéis nada mejor que hacer, escribirme, y si tenéis algo que hacer, dejarlo para luego, que ya habrá tiempo.
Los cambios respecto al numero anterior se pueden apreciar a primera vista. Una interface mejorada con Frames, un poco de java en la parte de arriba, y un poco de música XM, eso si, para escucharla tenéis que instalar el Plugin que se encuentra en el directorio \Modp dentro del directorio datos.
En cuanto a las secciones, solamente desaparecen tres, Refranes, y el curso de creación de mapas con Qoole y programación de Pascal. Pero tranquilos, estas dos ultimas secciones las retomare el numero que viene y con ración doble. Pero no solo desaparecen, también aparecen muchas mas. Por ejemplo, ahora tenemos un curso de Autocad, una introducción al Hacking, una curso rápido de como bloquear ordenadores a través de un Chat, una sección donde analizo las ultimas publicaciones electrónicas aparecidas en los últimos meses, un foro que también funciona como consulta, y algunas mas. Como veis, la calidad a subido bastante comparado con el numero anterior, y espero que siga así gracias a vuestras colaboraciones. Bueno, después de esta introducción ya podéis empezar a leer la revista.
