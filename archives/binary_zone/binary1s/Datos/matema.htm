Si eres un amante de los problemas matemáticos, esta es tu sección. Cada numero tendras un nuevo problema para resolver, y si encuentras la solucion mandamela. Si nadie me envia la solucion lo daremos por "no solucionado", pero el de este numero es muy facil, asi que a pensar.

Supongamos que tenemos dos números a y b, con a > b; entonces podemos expresar:

a = b + c : c > 0

Si multiplicamos ambos miembros por (a - b), obtenemos:

a · (a - b) = (b + c) · (a - b);

Y desarrollando seria:

a² - ab = ba - b² + ca -cb;

o lo que es lo mismo:

a² - ab -ac = ba - b² - cb;

sacando factor común:

a · (a - b - c) = b · (a - b - c)

y dividiendo entre (a - b - c), obtenemos:

a = b
