NOX

Presiona F1 durante el juego para activar la consola de trucos. Una vez abierta escribe:
CHEAT ABILITY: pone a 0 (cero) las habilidades del jugador.
CHEAT GOTO XXX: salta a un punto del juego.
CHEAT HEALTH: recupera la salud.
CHEAT MANA: recupera maná.
CHEAT LEVEL # : avanza hasta el nivel que marquemos.
CHEAT SPELLS # : sube el nivel de los hechizos hasta el nivel que le marquemos.
CHEAT GOLD # : pasamos a poseer la cantidad de oro que marquemos.
HELP CHEAT: listado de todos los trucos disponibles.
SET GOD: modo Dios y maná ilimitado.
