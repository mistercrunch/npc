Listas de definiciones

Las listas de definiciones pueden utilizarse, por ejemplo, para crear descripciones referentes a diferentes elementos. Los elementos de las lista de definiciones son las siguientes:

<DT>: seria el nombre de un termino
<DD>: la definición del termino anterior.

Ahora un ejemplo:

<DL>
<DT>Binary Zone<DD>La mejor revista que has leído nunca.
<DT>HTML<DD>Lenguaje de programación en para paginas web.
<DT>Windows 98<DD>Versión inestable de windows 3.11
</DL>

Y todo esto en una web, como esta, se representaría de la siguiente forma:



Binary Zone
La mejor revista que has leído nunca.
HTML
Lenguaje de programación en para paginas web.
Windows 98
Versión inestable de windows 3.11



Tablas

Las tablas son unos de los elementos mas útiles que dispondremos para la creación de paginas web. Estos son los elementos que podremos utilizar:

<TABLE>: Identificador de tabla.
<TR>: determina una fila en la tabla.
<TH> y <TD>: dos tipos de celdas en los que se puede dividir una fila.

Y ahora como siempre, un ejemplo para entender su uso.

<TABLE>
<CAPTION> Una tablita de ejemplo</CAPTION>
<TR>
<TD>Esto<TD>es una<TD>tabla<TD>
</TABLE>

Y así quedaría en nuestra pagina web:


Una tablita de ejemplo

Esto

es una

tabla
