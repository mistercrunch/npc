Bien, continuamos el curso de HTML donde lo dejamos en el numero anterior. En esta tercera parte se tratara un tema de vital importancia para nuestra futura pagina Web. ¿Que seria de una Web sin ningún gráfico? El visitante a primera vista no se vera muy atraído, aunque la información que contenga sea lo mas interesante del mundo, por eso necesitamos que la primera impresión sea lo mas impactante posible, y para ello necesitamos incluir gráficos.

Los tipos de archivos mas utilizados en las paginas Web son los (normalmente) imágenes Gif y Jpg. Para introducirlos en nuestra web contamos con la etiqueta <img> de cuyo modo de uso pasaremos a ver con un ejemplo.

<img src="binary.gif" ALT="Tu magazine favorita">

Mediante esta linea de código incluiremos una imagen llamada binary.gif. Pero ademas de la etiqueta img nos encontramos con algunas cosas nuevas, por ejemplo. SRC es un atributo mediante el cual indicamos cual es la imagen que deseamos que se visualice y con el otro atributo, ALT, ofreceremos un texto descriptivo, que será lo que vera el usuario que utilice un navegador sin capacidad gráfica (¿todavía existen?) También será el texto que aparecerá cuando coloquemos el cursor encima de una imagen.

Otros atributos de interés para aplicar a las imágenes son los siguientes:

ALIGN= left | right | top | middle | bottom
Left y right sirven para situar la imagen al margen izquierdo y derecho respectivamente.
Top, middle, bottom en cambio, se utilizan para situarla en la parte superior, media e inferior respectivamente con la linea de texto que se encuentre en elemento <img>

BORDER=x
Con este atributo enmarcamos la imagen con un borde de grosor x. Para los que sean muy cortos, en vez de x debéis de poner un numero, que será, como ya he dicho antes, el grosor del borde.

WIDTH=x, HEIGHT=x
Gracias a este atributo podemos especificar la anchura y altura de la imagen.

VSPACE=x, HSPACE=x
Se utiliza para indicar el espacio vertical y horizontal que queremos dejar entre la imagen y el texto que lo rodea.

Como podéis observar es muy fácil tener el control completo sobre la apariencia que tomara la imagen en nuestra pagina web. Si por ejemplo deseamos incluir un archivo llamado prueba.jpg, que tenga un borde de valor 2 y que tenga un espacio entre el texto y la imagen de 10, solo tendremos que escribir lo siguiente:

<img src="prueba.jpg" border=2 vspace=10 hspace=10>
