Bueno primero decir que son los sistemas NETxus:

Los sistemas netxus son una herramienta de gestión y transmisión de ficheros entre una red local y una red de ordenadores remotos (PC's, portatiles..).
Los sistemas netxus estan formados por 2 modulos:
Un modulo servidor, que se instala en el servidor de la red local y un modulo cliente que se instala en el terminal remoto.

Hay 3 tipos de sistemas netxus (aunque actualmente el netxus400 no se encuentra disponible en la web de la empresa)

a) Netxus 95

Este es el pequeño de la gama, ya que no tiene tantos requerimientos como el NETxus one call.
El módulo servidor de los Netxus 95 puede instalarse en un Windows 3.x/9x de Sistema operativo.
El módulo cliente se instala en Windows 3.x/9x
Esto es lo normal, pero tambien existe la posibilidad de utilizarse en equipos unix y  as/400. 

b) Netxus one call

El Netxus one call parece un sistema más interesante ya que incluye la posibilidad de  utilización de la telefonia gsm para conectar a la red local, conexión a la intranet, o a internet mediante infovia+.
El módulo servidor del netxus one call suele ser un Windows NT 4.0 y con Service pack 3 como minimo.
Los sistemas operativos de donde este instalado el modulo cliente pueden ser: 
WindowsNT Server/Workstation con Service pack 3 como minimo
Windows 9x

Al igual que en los Netxus 95 se puede instalar el módulo server en sistemas unix y as/400 mientras permitan el acceso a servidor ftp

C) Netxus/400 

Estos Netxus parecen haber sido sustituidos por la posibilidad de instalar los otros 2 tipos en sistemas as/400, unix.
El módulo servidor esta instalado en un as/400
