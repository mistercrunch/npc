------------B.T Revelation-------------------

GENERAL DESCRIPTION 
Small wall-mounted Hybrid/PBX system which comes in four sizes, 1+6, 2+6, 3+9, 4+12. The 1+6, 2+6, 3+9 may be extended in stages to the maximum capacity. The basic system is connected to analogue MF or LD exchange lines. There is an optional ISDN2 Access Unit which allows for alternative voice connections to either one or two ISDN2 Basic Rate Access connections. 

Numbering range: 20 - 31
Power Fail : Each line has one socket attatched
Exchange calling : LOOP
Approved Signalling : MF4/LD
Cleardown time : 100 mS

SOCKET WIRING:

IDC Wire Colour  LJU
A   Blue/white   2
B   White/blue   5
C   Green/white  1
D   White/green  6
X   Orange/white 3
X   White/orange 4

100 ohms or 700 metres for Systemphone extensions.
200 ohms or 1400 metres for approved telephone extensions.
336 ohms or 2 kilometres for the 2 wire external extension.

ADMINISTRATION PROGRAMMING 
From the programming extension systemphone.
Do not lift the handset.

Enter the access code
Enter the access PIN The display shows the prompt Dial admin code. 
Enter the required admin code.
Enter any required data.
To enter another admin code, step back to the Dial admin code prompt by pressing #. Some features require you to press # more than once.
To finish programming, press HANDS FREE
