SECtion

Alimentation requise
   Le B1225 opère à partir d'une source d’alimentation de +11.5VDC à +28VDC. 
100mA plus 10mA par relais

Installation – Alarmes
   Dans les conditions par défaut, le B1225 regarde pour un `Normalement 
Ouvert NO` pour désactiver et pour un `Normalement Fermé NC` pour activer une 
alarme avec le ground. Ceci peut être modifié dans les configs I/O, format 
d’alarme. Toutes les entrées d’alarmes sont opto-isolées (sorte de relais 
isolé par procédé optique) et opère avec le 12v interne.

Caution : DO NOT connect a voltage source to the alarm inputs.

Radio
   Si une alarme à besoin d’être annoncée par le biais de fréquence radio, 
connecter à la borne TX à la borne de transmission audio du radio. 
L’output PTT fournit la masse GND par un collecteur ouvert pour signaler 
au radio de transmettre. À relier au Push-To-Talk Input du radio.

   L’entrée COS est utilisée pour indiquer au B1225 que le canal radio est 
occupé. 
À connecter à la sortie Carier-operated switch ou relay COS ou COR. 

Accusé de réception
   Cette entrée est utilisée dans le but d’accusé de réception de l’alarme 
localement sur le B1225 sans avoir à appeler sur fréquence ou sur une ligne 
téléphonique.

Séquence de démarrage
   Une fois l’installation complétée, l’alimentation DC pour être branchée
pour l’alimentation du circuit. Au démarrage le B1225 passe par une séquence 
d’auto-test (ex ; s’assurer qu’il y ait toujours 10 doigts sur les mains en se 
réveillant le soir ou le matin):

1 LED test
	Voice, Tone and , Hook Leds On
	PTT Led Off
	Run Led Flashing
2 Hardware Test
	Alarm Leds 1-8 On
	Memory test
	Une seconde pour le test  
3 Vocabulary voice test
	Alarm Leds 1-7 On
4 User Voice Message test
	Alarm Leds 1-6 On
5 Database setup
	Alarm Leds 1-5 On
6 End of test
	Alarm Leds 1-8 Off
7 Operation normale
	Run led Steady
