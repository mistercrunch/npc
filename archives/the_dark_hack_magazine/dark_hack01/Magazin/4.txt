Chanserv

/msg chanserv register #canal password descripcion : registra un CANAL

/msg chanserv drop #canal : es para dar de baja un CANAL

/msg chanserv identify #canal [password] : identifyaca un CANAL

/msg chanserv access #canal add [nick] [nivel] : para dar access

/msg chanserv access #canal del [nck] : para sacarlo de las lista de access

/msg chanserv unban cana nick : por si te banean

/msg chanserv info #canal: info de CANAL

/msg chanserv set #canal [pass new] : para cambiar la contra de tu CANAL

/msg chanserv set #canal entrymsg [mesaje] :menaje en el CANAL

/msg chanserv chan set founder canal nick : cambia de dueño

/msg chanserv set password nueva_pass : para cambiar la contra de tu CANAL

 

Nickserv

/msg nickserv identify [pass] : identifyca el nick

/msg nickserv drop [pass] : si te arrepentiste de reg. tu nick

/msg nickserv recover [nick] [pass]: si te utilizan el nick

/msg nickserv info [nick] : ves como esta configurado el nick

/msg nickserv set password [password] : cambia el pass de nick
