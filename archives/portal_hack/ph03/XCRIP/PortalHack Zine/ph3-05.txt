Muchos celulares tiene el ESN/MIn en una Eprom/EEprom ubicada en algun
lugar del circuito.
Cuando prendemos el celular este envia el ESN/MIN a la celular en el 
*Reverse Channel*. La celula verifica en su base de datos si esta autorizado. 
Si todo esta bien  el celular podra realizar una llamada.

*Reverse Channel*Los datos que pasan por aqui (los del celular) pueden ser 
capturados. Explicacion mas adelante

Algunos phreakers cambiaron el software de la eprom, logrando que automaticamente
el celular capture ESn/MIN y los introduzca automaticamente.

como hemos visto antes, pondremos los codigos de la Marca celular, para
acceder al seteo, y obtener el ESN y el MIN (LEER PARTE I).

2. Pinchando lineas ;))

Esquema para pinchar lineas telefonicas

La linea se puede pinchar en dos lugares diferentes, en el cable que va
de la casa a la caja donde se juntan las lineas o bien en dicha caja.

Si lo conectamos en el cable no habra problemas de identificacion pero
dara mucho encanto ver un aparato pegado al hilo.
Si lo conectas dentro de la caja no se vera (a menos claro que los de la 
empresa telefonica revisen la caja) pero puede haber problemas para 
identificar el numero es asi, llevate un movil GSM y un telefono normal 
conecta este a cada linea y llama al movil, el Caller-ID te permitira saber 
cual el la correcta.

Este es el esquema para pinchar una linea telefonica.

o---||---\/\/\-------+--------\/\/\-------+---------+
    C1     R1        |          R2        |         |
                   +-+-+                  |         |
                   |   |                  /         |
                  __   __                 \         +---- ALTAVOZ/
LINEA          D1 /\   \/ D2              /
                  --   --              R3 \         +---- GRABADOR
                   |   |                  /         |
                   +-+-+                  |         |
                   | | |
o--------------------+--------------------+---------+

Los componentes son:


C1: 47nF
R1: 27K
R2: 4K7
R3: 120
D1, D2: 1N4148

El grabador tiene que ser del tipo de los que se activan por la voz.

C1 elimina la continua y evita la elevada tension del tono de llamada (mas
de 100 voltios); R1, D1 y D2 recortan la señal alterna a algo menos de
1 voltio; por ultimo, el divisor R2, R3 reduce la señal para evitar que
el grabador se dispare cuando haya ruido en la linea.
