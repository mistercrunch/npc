Como modificar los créditos de autor del programa Internet Explorer 4.0
________________________________________________________________________

La manera en que podemos modificar el contenido de la animación que se nos presenta en los créditos de autor del programa Internet Explorer 4.0, consiste en hallar el archivo que contiene oculto dentro de sí, el código fuente del documento HTML que se activa en el momento en que damos click en el botón "Desbloqueado" de la opción "Acerca de Internet Explorer" del menú "Ayuda".
______________________________________________________________________

NOTA: Obviamente esto lo hacemos mediante el huevo de pascua del Internet Explorer 4.0, si no saben como se hace es bueno que lo aberiguen, o esto no serviria de nada.
______________________________________________________________________

Indagando atraves de las librerías que se relacionaban con la ruta de acceso y nombre de archivo de otros documentos que son utilizados por el Internet Explorer, para mostrar información en pantalla, encontré una librería de nombre Shdocvw.dll en la ruta de acceso C:\Windows\System\ con la propiedad de archivo oculto, luego de abrirlo con un editor de texto y de ver unas cuantas líneas cifradas en su contenido, más abajo encontré lo que buscaba el código fuente del documento HTML que se activa en la animación.

En su interior utilizaban una curiosa función programada en JavaScript para cifrar los nombres y títulos que aparecen en pantalla, no siendo un inconveniente puse ha trabajar la función al revés, obteniendo las letras reales y su equivalente cifrado, como lo pueden apreciar en la

Función encargada de cifrar los valores que contiene el arreglo
______________________________________________________________________

var g_nEncodeOffset = 13 ;
var g_nEndOfLineEncoding = 123 ;
var g_nBeginningOfCharSet = 65 ;
var g_nCharSetSize = 52 ;
var g_nStartOfUpperSet = 90 ;
var g_nSetDelta = 6 ;
var g_nSpace = 32 ;
______________________________________________________________________

function DecodeChar(nIn) // nIn es el argumento de la función que contiene el
{   // valor ascii de una letra contenida en el arreglo
var nOut; // define la variable nOut
if(nIn == g_nSpace) // si el valor nIn es igual a 32 [valor ascii de espacio ]
return " "; // retorna un espacio

else if(nIn == g_nEndOfLineEncoding) // de lo contrario si nIn es igual a 123 [valor ascii ({)]
return String.fromCharCode(13,10); // retorna un salto de linea
else // de lo contrario

{
if(nIn > g_nStartOfUpperSet) // si nIn es mayor que 90 [valor ascii (z)]
nIn = nIn - g_nSetDelta; // nIn es igual a nIn menos 6

nOut = (nIn - g_nEncodeOffset); // nOut es igual a nIn menos 13 [26 / 2]

if(nOut < g_nBeginningOfCharSet) // si nOut es menor que 65 [valor ascii (a)]
nOut = (nOut + g_nCharSetSize); // nOut es igual a nOut mas 52
if(nOut > g_nStartOfUpperSet) // si nOut es mayor que 90 [valor ascii (z)]
nOut = nOut + g_nSetDelta; // nOut es igual a nOut mas 6
}

return String.fromCharCode( nOut ); // retorna el caracter equivalente al valor ascii de nOut

}
______________________________________________________________________
