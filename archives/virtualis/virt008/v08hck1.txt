  INTRODUÇÃO
  ----------
  
  Com certeza voce já entrou em uma página e leu bem em destaque na
  página "bem vindo.200.210.xxx.xxx"... poxa como eles sabem seu IP ?
  Você sabe que com seu IP um hacker pode fazer algumas coisas...
  Mesmo que voce desabilite o java e java script seu IP pode ser 
  logado no contador do site... e agora ???

  A solução é o proxy !!

  Proxy não tem nada a ver com cookies, filtro de java e nada disto...
  Os servidores acham seu browser, IP, sistema operacional, não é
  através do cookie ou Java ou ActiveX ... Estas informações seu 
  computador entrega de bandeja para TODO servidor devido ao que 
  chamam de HTTP HEADER... isto faz parte do http...

 
  COMO FUNCIONA
  -------------
  
  Vamos entender o que o proxy faz que te deixa anônimo. O proxy é um
  depósito de home pages para uma rede interna de alguma empresa (ou
  provedor); entre outras possiveis funções. Quando você se conecta a
  um proxy e digita um endereço de página, seu browser pede a página
  ao proxy...se ele tem em seu depósito ele te envia direto... se ele
  não tem em seu depósito ele pede a página para o endereço que você
  digitou... e guarda no depósito após te enviar uma cópia.

  Ex: você digita www.yahoo.com o browser grita para o proxy "Salta a
      página do Yahoo!!"
      
  O proxy procura na dispensa de paginas... não tem mais... então ele
  vai no www.yahoo.com e copia a página pedida. Manda uma cópia para
  você e guarda uma cópia para futuros pedidos de alguém. Ele é um
  grande cache... como o que seu browser faz.
  
  Deste modo é facil entender como voce fica anônimo. Você fica
  anônimo porque não foi você que acessou a página do Yahoo, não foi
  seu IP que conectou ao servidor do Yahoo, então ele não foi logado.
  Foi tudo feito pelo proxy... por isto também que fica mais lento...
  Acho que fui claro...


  COMO SABER SE O PROXY É ANÔNIMO?
  --------------------------------
  
  Esta é um boa questão...
  
  Visite um dos sites abaixo:

    http://internet.junkbuster.com/cgi-bin/show-http-headers
    http://anonymizer.virtualave.net/cgi-bin/env.cgi
    http://cserve.cis.smu.edu/misc/envtest.html
 
  Estes são scripts CGI que mostram o que aparece nos HTTP HEADER.
  Dê uma olhada nos seguintes itens:
  
    REMOTE_ADDR: seu ip
    REMOTE_HOST: seu host
    HTTP_X_FORWARDED_FOR: proxy mostra o IP de quem o utiliza (dedo-duro)
    HTTP_USER_AGENT: seu browser
    FORWARDED: mostra o proxy usado
    VIA: mostra uso do proxy
    CLIENT_IP: seu IP novamente revelado pelo proxy
    HTTP_FROM: seu IP novamente revelado pelo proxy
    
  Nem sempre todos estes itens são mostrados. Se algum destes itens
  revelar seu IP ou host... dançou... não é anônimo... Um proxy 
  excelente também oculta o seu Browser e sistema operacional. Mas
  estes são raros, e neste caso é melhor usar o Proxomitron.
