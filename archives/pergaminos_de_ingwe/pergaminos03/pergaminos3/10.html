ARMANDO UNA WEB CON HTML III

Después de ver un montón de maneras de escribir dentro de un documento HTML, vamos a ver otras cosas que son comunes a la hora de hacer una web.

IMAGENES

La etiqueta con las que van referidas las imágenes son <IMG SRC="nombre de la imágen" ALT="comentario" HEIGHT=nro. WIDTH=nro.>

En primer lugar veamos la primera parte: IMG SRC (image source) aca debemos indicarle la imagen con la extensión que vamos a poner.
ALT es opcional (como height y width) y lo que hace es poner un comentario al pasar con el mouse.
HEIGHT y WIDTH dan la altura y el ancho respectivamente.

Pongo como ejemplo este pequeño código y abajo muestro el resultado:

<IMG SRC="caritasaluda.gif" ALT="Esta carita podría ser la mascota de IngweSoft">

Bien, abajo tienen el resultado, posen también el mouse sobre la imagen

Evidentemente puedo agrandarla a gusto y combinarla con otras etiquetas, probemos lo siguiente:

<CENTER>
<IMG SRC="caritasaluda.gif" ALT="Así grande queda más feo que la mierda" HEIGHT=100 WIDTH=50>
</CENTER>

ENLACES

Las etiquetas utilizadas para hacer enlaces son <A HREF="destino"> y </A>

Por ejemplo, supongamos que quiero hacer un link al sitio de IngweSoft, podríamos poner la siguiente linea:

<A HREF="http://ingwesoft.tripod.com.ar">LINK A INGWESOFT</A>

Lo cual quedaría así:

LINK A INGWESOFT

Si lo que quisiera hacer es hacer un link a una dirección de mail, podría hacerlo de esta manera (ejemplo de link a e-mail de ingwesoft):

<A HREF="mailto:ingwesoft@yahoo.com.ar">ENVIAR MAIL A INGWESOFT</A>

Esto sería de esta manera:

ENVIAR MAIL A INGWESOFT

También pueden poner una imágen como link, probemos con este ejemplo:

<A HREF="http://ingwesoft.tripod.com.ar"><IMG SRC="caritasaluda.gif" ALT="Click para ir al sitio oficial de IngweSoft"></A>

Los links no solo pueden ser entre documentos, sino que se puede hacer dentro de este usando lo que llamaremos "anclas".
Las anclas se definen con la etiqueta <A NAME="nombre del ancla"> y a la hora de hacer el código del link a esa zona del documento HTML, lo haríamos poniendo:

<A HREF="#nombre del ancla">Texto o imágen del link</A>
