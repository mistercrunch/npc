Truquillos para echar partidas gratix (por Cacique, Jaime Robles y Blake)
-------------------------------------


 Para todos aquellos aficionados a las máquinas recreativas.
 El truco en custión es sacar partidas gratis de estas máquinas.

 A mi me ha funcionado con duros antiguos, si, de los grandes.
 La "cosa" está en coger y meter el duro en una ranurita que está
 en el hueco por donde se devuelven las monedas.


     ------------------
     |    ---   ---   |
     |    | |   | |   | -> Por aquí se meten las monedas normalmente.
     |    ---   ---   |
     |                |
     |    ---   ---   |
     |    | |   | |   | -> Por aquí salen cuando le das a devolver.
     |    ---   ---   |
     ------------------

 Pues en los huecos de abajo, si meteis el dedo doblado hacia arriba,
 palpareis unas especies de guias, es decir, dos plaquitas de metal, cuya
 distandia es casi la de un duro ( en realidad la obertura es mayor). Se
 trata de meter el duro antiguo entre esas 2 plaquitas.
 Antes de nada, probad a encontrar las 2 palquitas.
             -----------
                 | |
                 | |  -> Debeis palpar algo así dentro del hueco,
                 | |     ya sabeis, meteis el dedo y hacía arriba.
             -----------

 Espero que os salga.

 Otra forma es con un mechero electrico, sacar un par de cablecitos de conde
 saltan las chispas hasta el recuadro metálico que rodea el plástico de
 meter monedas. Accionas el pulsador, y suele dar partida gratix.
 Pero ojo, si lo que rodea al cajetin de meter las monedas, es de plástico,
 también, el truco no funcionará.

 Ya me contareis. Quizás sean trucos muy viejos, pero seguro que
 hay alguien que no los conocía.

 Perdon por los gráficos ;))


 Truquillos para Futbolines

 Pues muy simple..Cuando insertes la moneda, con un clip en forma de gancho
 dejas enganchado la corredera que se mueve al tirar del 'pomo' ¿entiendes?
 Así se queda abierta la caja de las bolas y cada vez que se mete una por la
 portería vuelve a caer.. Simple ¿no? 

 Otro truco...
 Se trata de colocar unas hojas de papel, unos simples folios, en las porterias,
 por dentro, impidiendo que la bola entre a los "canales" del futbolin, pero
 colocandolo de forma que no se vea desde fuera... asi, cada vez que se meta
 un gol, se mete la mano y se agarra lapelota ;-)
