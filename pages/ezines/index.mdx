# Other Ezines

Historical underground ezines from the BBS era, preserved and converted for the modern web.

## Available Ezines

- [NoWay](/ezines/noway) - First French hacker zine (1994-1997), Minitel hacking, cyberpunk culture
- [40Hex](/ezines/40hex) - Virus writing zine by Phalcon/Skism (1991-1998), 14 issues

## Adding More

To add another ezine from the [autistici.org archive](https://www.autistici.org/ezine/):

```bash
node scripts/fetch-ezine.mjs <ezine-name>
```

Example:
```bash
node scripts/fetch-ezine.mjs 40hex
```
