# Ezine Archive

Historical underground ezines from the BBS era (1980s-2000s), preserved and converted for the modern web.

**Source:** [autistici.org/ezine](https://www.autistici.org/ezine/)

## Available Ezines

- [40Hex](/ezines/40hex) - Virus writing zine by Phalcon/Skism (1991-1998)
- [CUD](/ezines/CUD) - Computer Underground Digest
- [FrHack](/ezines/FrHack) - French hacker zine
- [LOD](/ezines/lod) - Legion of Doom Technical Journals
- [NoWay](/ezines/noway) - First French hacker zine (1994-1997)
- [Phrack](/ezines/phrack) - The legendary hacker zine (1985-2016)

## Adding More

```bash
node scripts/fetch-ezine.mjs <ezine-name>
```
