}{========================}{PARTICIONANDO EL DISCO RIGIDO}{========================}{


Para poder instalar adecuadamente este SO tenes que crear 2 particiones, una LINUX NATIVE y otra LINUX SWAP. Entonces teniendo el HD ya formateado metemos el Disco de arranque de Win (si es que antes tenias win instalado o si vas a usar otro HD para instalar linux) y booteamos desde el Diskette. Ponemos 'FDISK' y creamos las particiones
Ej: Suponiendo que tenemos un HD de 10 gb y una RAM de 32 debemos particionar el disco de la siguiente manera:

Part 1 .....................9.800 MB ------>Esta particion la setearemos despues como NATIVE
Part 2 ..................... 200   MB-------> Y esta sera la particion SWAP

Despues de particionar el disco correctamente y guardar los cambios, ya estaras listo para empezar a instalar LINUX SLACKWARE

}{=============================}{BOOTEANDO DESDE EL CD}{==========================}{

Lo primero que hay que hacer para la instalacion es setear la BIOS de manera tal que bootee primero desde el cd-rom. Para lograr esto encendemos la PC y ni bien termina de chequear la Memoria apretamos la tecla 'suprimir'.
Haciendo esto accederemos al setup de la maquina, ahora entramos en "Advanced CMOS" [esto depende del modelo de BIOS que tengas]. Una vez adentro solo te resta modificar el orden de booteo de la maquina, te vas a dar cuenta porque dice: 

1║ Boot : IDE
2║ Boot : FOPPY

Aca tenes que cambiarlo posicionandote encima con el cursor y apretar la tecla  [+] o [-] de tu teclado.Una vez echo esto le damos 'escape' y ponemos SAVE CHANGE AND EXIT, le damos YES y listo.
Con esto se reseteara tu PC, metes el CD de instalacion de Slackware y listo, ya estamos en la instalacion :-)


}{=============================}{STEANDO LAS PARTICIONES}{==========================}{

Despues de elegir el kernel ya estamos listos para crear las particiones, ponemos 'cfdisk' y entramos al programa. Ahora seleccionamos la particion 1 [la de mayor tama±o] y dale enter posicionando el cursos en el menu inferior [Type]. Ahi elegimos el numero 83 [LINUX NATIVE] y ya quedara seteada la particion como NATIVE, ahora hacemos la misma con la otra particion pero seleccionamos el numero 82 [LINUX SWAP].
Escribimos los cambios y listo. Ya estan creadas las dos particiones necesarias para instalar linux

}{====================================}{INSTALANDO}{================================}{

Tipeamos la palabra 'setup' y le damos [ENTER], con esto accederemos al menu de instalacion de Slack.
Una vez adentro del menu tenes q ir a detectar swap, detectarß tus particiones e iniciara la instalacion. Solo te resta elegir el tipo de Instalacion. Yo recomiendo la 'MENU'. Despues de elegir el modo de instalacion empezara la carga de archivos, este proceso puede tardar entre 2 y 3 horas.
Antes de instalar los grupos de paquetes te pregunta si los queres instalar o no, si es la primera vez q instalas linux te recomiendo q instales todos los paquetes.
Te dejara instalar 2 kernels, Apache, gcc, MySQL, KDE, GNOME, etc etc...muchisimas aplicaciones :-))

IMPORTANTE: Tambien te pedira q setees tu login y tu password de root [si es posible trata de q sea un poco larga, alfanumerica y facil de recordar por ti]

}{=============================}{CONFIGURANDO ACCESORIOS}{========================}{

Despues de que termino la instalacion de todos los grupos de paquetes [packages] te aparecera una ventana para configurar tus perifericos. Tenes que configurar el puerto donde esta instalado el mouse, el puerto donde esta el modem [COM1-COM2-COM3-COM4], El tipo de teclado, y la zona horaria.
Esta parte realmente es muy facil de hacer, no creo q tengas inconvenientes.

Tambien tendras q elegir en que resolucion queres trabajar y que gestor de ventanas queres utilizar. Tenes varias alternativas como son KDE , GNOME y WM, te recomiendo q elijas KDE, es mas facil!

}{=================================}{INSTALANDO LILO}{===============================}{

Una vez echos los pas0s anteriores tendras que instalar el LILO [LInux LOader], este programita es el que te permitira elegir que sistema operativo queres arrancar [en caso de q tuvieras mas de uno ej:WINDOWS 98 y LINUX SLACKWARE]
Puedes elegir varias opciones 'en donde instalar lilo' yo te recomiendo 2:  lo instalas en un diskette, asi cada vez que booteas del DISKETTE arrancara lilo y te preguntara q sistema oprativo queres correr; o lo instalas en MBR [master boot record], esto lo q hace es instalarlo en tu HD y cada ves q inicies la PC arrancara lilo y te dejara elegir los SO`s.

Cualquiera de las dos opciones se generan sin complicaiones, no tendras problemas con esto :-)

}{======================}{LEVANTANDO X Y EL GESTOR DE VENTANAS}{=====================}{

Una vez configurado todo, se reseatara y te pedira q te loguees, tenes q poner lo siguiente:

LOGIN: root
PASS  : tu contrase±a de root
Y le das [ENTER]

Si escribiste todo bien, cargara el kernel y ya estas listo para levantar el servidor X y el gestor de ventanas. Para lograr esto debes tipear 'kdm' y se cargara KDE, te pedira login y pass, y listo!! Ya estas listo para empezar a trabajar con este sensacional Sistema Operativo.

Si solo quieres levantar el servidor X, debes tipear 'xinit' o 'startx'

*** SI NO SE VE BIEN LOS GRAFICOS O SE VE CORTADO, O TE TIRA ERROR, es muy probable que tengas que configurar tu monitor y placa de video. Para hacer esto debes tipear 'xfreesetup' (antes de poner KDM) y darle enter, ahi configuras de manera sencilla tu monitor. Si el problema continua te recomiendo que pongas 'xf86config' , ahi podras setear la frecuencia del monitor verticalmente y horizontalmente, la configuracion, la placa de video, tu mouse y tu teclado. Hasta tambien el idioma podras setear.
