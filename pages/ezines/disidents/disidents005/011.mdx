using System;

public class p12 {

public static void Main () -------> Estas tres lineas se da el nombre a la clase 
y con el { se abre el bloque del codigo

leer informacion desde el teclado osea que el usuario introduce la informacion.
 La orden Parse lo que hace es convertir lo que escribes por teclado en " float " ( dato numerico )

|===========~ 5- Estructuras especiales. ~|=================================
|===========================================================================
 
Estas estructuras aumentan la potencia del lenguaje al simplificar asi muchos 
metodos y facilitar la resolucion de problemas la primera estructura que veremos es:

		\\ IF .... ELSE //

Esta estructura se ejecuta condicionalmente, es decir, si una condicion es 
cierta se ejecuta una cosa y si no otra. Hay dos resultados posibles: cierto o falso.

Ej.

	if (dia==22)
	{
	    Console.WriteLine("El dia de hoy es dia de fiesta");
	}
	else
	{
	    Console.WriteLine("El dia de hoy es dia laboral");
	
	}


Analicemos linea a linea la sintaxis.

if (dia==22) ----> if (si... ) Si la variable o constante " dia " es 22 se cumple la accion siguiente delimitada por {}

else -----> Si esta variable o constante no es igual a 22 se ejecuta la accion siguiente al else ( si no... )


	Esta estructura es muy utilizada y se pueden coordinar tantos 
	if...else como se quieran, incluso unos dentro de otros.

_____________________________________________________________________________
****NOTA IMPORTANTE****

Hay que tener muy encuenta que hay que cerrar cada bloque con {} un consejo 
que doy es ir poniendo cada corchete que abre a la altura del que lo cierra :)

____________________________________________________________________________


		\\ SWITCH //

Esta estructura es parecida al if...else y tambien es de condicion.Veamos un ejemplo.

switch (eleccion)
{
    case 1:
        precio=10;
        break;
    case 2:
        precio=5;
        break;
    case 3:
        precio=2;
        break;
    default:
        precio=1;
        break;
}


Analizando la estructura vemos que:

Si la eleccion que se da en la primera linea switch es un caso de los abajo 
descritos por case1, case2... se realizara la accion seguida precio=x,esto se 
haria mejor con un if...else pero tambien puede resultar util. El break solo
es para que el compilador al llegar a ese punto se detenga si no, al darse 
el caso 2 luego seguiria al 3 y asi continuamente entonces con " break " se acaba la orden.



		\\FOR//

Esta instruccion sirve para crear un bucle ( orden que se repite x veces ) y 
tambien simplifica mucho trabajo. Veamos la sintaxis.

for (valor inicial de la variable;condicion;valor posterior)
{
    Instrucciones
}


Esto se leeria de la siguiente forma, hasta que la variable no alcance la 
condicion se ira ejecutando, tomando la variable un valor distinto al hacer cada recorrido de la orden.

Por ejemplo:

for ( dia=0;dia<15;dia++ )
{
	Console.Write(" Estamos en la quincena primera de mes " );

}

La orden dia++ sirve para que cada vez que el programa ejecute esa orden 
sume 1 a la variable dia que principalmente vale 0, entonces recorrera 15 veces 
el bucle hasta alcanzar la condicion 15 diciendonos asi ke es la primera quincena del mes.


		\\WHILE//

Esta estructura crea un bucle mientras no se cumpla la condicion.

var= 1 ----> Declaras la variable
While ( var = adi ) ----> Estructura While con la condicion entre ()
{
Console.Write("Adivina el numero"); ---> Te " pregunta " el numero jeje :)
adi=Console.ReadLine(); ----> Asignale el valor a adi para que sea igual a var y se cumpla la condicion
}
Console.Write("Correcto"); ---> Ha salido del bucle y te escribe en pantalla " Correcto "


Declaro la variable " var " y le asigno de valor 1, la condicion es que las 
variables var y adi sean iguales. Esto te pedira el valor de adi hasta que sea 
el mismo que el de var y se cumpla esa condicion, entonces saldra del bucle y saldra por pantalla " Correcto".
