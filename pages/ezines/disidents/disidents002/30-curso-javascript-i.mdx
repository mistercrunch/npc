------------[1.0 INTRODUCCION ]------------------------------------------------------------------
-------------------------------------------------------------------------------------------------


 Lo primero es explicar una breve introduccion de JavaScript. Es una de las multiples aplicaciones 
 que han surgido para extender las capacidades del Lenguage HTML.JavaScript es un lenguaje script 
 orientado a documento, nunca podras hacer un programa con este lenguaje, tan solo podras mejorar 
 tus paginas web =).





-------------------------------------------------------------------------------------------------
------------[2.0 NORMAS DEL CODIGO ]-------------------------------------------------------------
-------------------------------------------------------------------------------------------------


 Las normas para poder escribir cualquier codigo de JavaScript se basan en 5 puntos basicos y que 
 debemos cumplir siempre. Las normas son las siguientes ;)

     1. Todo el codigo(sentencias) esta dentro de las funciones.

     2. Las funciones se desarrollan entre las etiquetas &lt;SCRIPT> y &lt;/SCRIPT>

     3. Las etiquetas &lt;SCRIPT> deben colocarse entres las etiquetas &lt;HEAD> y &lt;/HEAD>

     4. Las etiquetas &lt;TITLE> no pueden estar entre &lt;SCRIPT> y &lt;/SCRIPT>

     5. La llamada a la funcion se hace a traves de un evento de un elemento del documento.






-------------------------------------------------------------------------------------------------
------------[3.0 USO DE LAS FUNCIONES ]----------------------------------------------------------
-------------------------------------------------------------------------------------------------

 Las funciones son un conjuto de sentencias(bloque de codigos) que especifica al programa las 
 operaciones a realizar. Son utiles para evitar la repeticion de lineas y moderar el codigo. Para 
 trabajar con ellas hay que desarrollarlas y llamarlas cuando lo necesitemos.

          Sintaxis del desarroyo:
                            
                            function nombre_funcion(var1,var2,var3...)
                            {
                                     sentencia(S);
                            }

En este caso la llamada se realiza desde un elemento del documento.

          Sintaxisde la llamada:
                            &lt;elemento evento=nombre_funcion(val1,val2,val3...);>
                            nombre_funcion(val1,val2,val3...);

Y en este otro caso la llamada se realiza desde el interior de otra funcion que tambien es 
posible.

 Se trata de la ventanita estandar que usa el win2 para mostrar una informacion en la pantala xD. Se

------------[5.0 NUESTRO PRIMER EJEMPLO ]--------------------------------------------------------
-------------------------------------------------------------------------------------------------


 Ahora para que esto no sea tan lioso vamos a hacer nuestro primer programa paso a paso, y asi 
podremos ver los elementos principales del lenguaje y su colocacion dentro del documento de HTML.
Solo debemos  seguir la teoria vista en los temas anteriores.

 EJEMPLO 1: Llamada de una funcion desde del documento.

     &lt;HTML>
     &lt;HEAD>
     &lt;SCRIPT>
     function hola()
     {
     alert("hola komo tamos!");
     }
     &lt;/SCRIPT>
     &lt;TITLE>titulo&lt;/TITLE>
     >/HEAD>
     &lt;/HTML>


 EJEMPLO 2: Llamada a una funcion desde otra.

     &lt;HTML>
     &lt;HEAD>
     &lt;SCRIPT>
     function hola()
     {
     alert("hola como tamos!");
     rehola();
     }
     function rehola()
     {
     alert("hola de nuevo viejo!!");
     }
     &lt;/SCRIPT>
     &lt;TITLE>titulo&lt;/TITLE>
     &lt;BODY onLoad=hola();>
     &lt;/BODY>
     &lt;/HEAD>
     &lt;/HTML>
